# Speech and Debate Simulator
Uses ChatGPT and assets from Brick Project Studio(Building) and Bucket(Mannequins).

ChatGPT model trained on Model United Nation sample speeches and grades used to listen to student's speech and grade their speech. Model returns formatted letter grading sheet with feedback.
