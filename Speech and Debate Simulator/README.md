# Speech and Debate Simulator
Uses ChatGPT and assets from Brick Project Studio(Building) and Bucket(Mannequins).

ChatGPT model trained on Model United Nation sample speeches and grades. A recording of the students speech is processed by the model and a formatted letter grading sheet with feedback is returned. This sheet is then displayed for the student to review.
